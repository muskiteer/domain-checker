Domain Checker API

Simple serverless API to check domain DNS resolution and HTTP/HTTPS reachability.

Endpoint

POST /api/check-domain

Example Request

curl -X POST "https://domain-checker-gray.vercel.app/api/check-domain" \
  -H "Content-Type: application/json" \
  -d '{"domain":"example.com"}'

Response

{
  "domain": "example.com",
  "dns_resolves": true,
  "ip": "93.184.216.34",
  "http": {
    "ok": true,
    "status_code": 200,
    "url": "http://example.com/"
  },
  "https": {
    "ok": true,
    "status_code": 200,
    "url": "https://example.com/"
  }
}

Input

JSON body with domain field (string). The API will strip http/https prefixes and paths automatically.

Output

  • domain: cleaned domain name
  • dns_resolves: boolean indicating if DNS lookup succeeded
  • ip: resolved IP address (or null)
  • http: HTTP check result (ok, status_code, url, error)
  • https: HTTPS check result (ok, status_code, url, error)
